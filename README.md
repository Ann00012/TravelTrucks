# 🚐 TravelTrucks - Camper Rental Service

TravelTrucks is a modern web application designed for discovering and booking camper vans. The project focuses on a seamless user experience, professional UI/UX, and high-performance standards using the Next.js App Router.

## 🚀 Demo
[Link to your live site on Vercel]

## ✨ Features

- **Camper Catalog**: Browse a wide range of campers with detailed specifications.
- **Dynamic Filtering**: Filter campers by location, equipment (AC, Kitchen, TV, etc.), and vehicle type (Van, Fully Integrated, Alcove).
- **Detailed Pages**: Individual routes for each camper with images, reviews, and technical features.
- **Booking System**: Integrated booking form with validation to reserve a camper.


## 🛠️ Tech Stack

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [CSS Modules](https://github.com/css-modules/css-modules) 
- **Notifications**: [React Hot Toast](https://react-hot-toast.com/)
